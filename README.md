Mark Down语法

1. 标题
使用 # 符号表示标题，数量表示级别（最多六级）：
# 一级标题
## 二级标题
### 三级标题

2. 文本样式
粗体：使用 **文本** 或 __文本__
斜体：使用 *文本* 或 _文本_
粗斜体：使用 ***文本***
删除线：使用 ~~文本~~
下划线：使用 HTML 标签 <u>文本</u>

3. 列表
无序列表：使用 -、* 或 + 开头
有序列表：使用数字加英文句点（如 1. ）开头
任务列表：使用 - [x]（已完成）或 - [ ]（未完成）
嵌套列表：在子列表项前添加四个空格即可

4. 链接与图片
超链接：[链接文字](URL)
图片：![替代文本](图片URL)

5. 引用与分隔线
引用：使用 > 开头，支持嵌套（如 >>）
分隔线：使用三个或以上的连字符 ---、星号 *** 或下划线 ___

7. 代码
行内代码：使用反引号包围，如 console.log()
代码块：使用三个反引号包围，并可指定语言以实现语法高亮（如 ```python）

8. 表格
使用 | 分隔不同的单元格，使用 - 分隔表头和其他行，可通过冒号 : 控制对齐方式：
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 单元格 | 单元格   | 单元格 |

8. 转义字符
如果不想让特殊字符（如 *、#、[] 等）触发 Markdown 格式，可以在字符前添加反斜杠 \ 进行转义。

9. 高级/扩展语法
脚注：使用 [^1] 标记，并在文末使用 [^1]: 脚注内容 定义
数学公式（部分编辑器支持）：行内公式使用 $公式$，块级公式使用 $$公式$$
内嵌 HTML：Markdown 兼容 HTML 标签，可直接在文档中编写 HTML 代码
